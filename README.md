## Tech Stack

- **Frontend**: Next.js 14, React 18, TypeScript
- **Blockchain**: Stacks Mainnet
- **Smart Contract**: Clarity
- **Libraries**: @stacks/connect, @stacks/transactions, @stacks/network

## Contract Functions

- `stake` - Stake STX tokens
- `unstake` - Withdraw staked tokens
- `claim-rewards` - Claim accumulated rewards
- `get-stake` - Get user stake info
- `calculate-rewards` - Calculate pending rewards
